# SF Restaurant Rank

Live demo: [https://sf-restaurant-rank.vercel.app](https://sf-restaurant-rank.vercel.app)

A head-to-head San Francisco restaurant ranking site with:

- ELO-based voting
- Live rankings and 7-day trend movement
- Category, price, and location filtering
- A large Overture-backed SF restaurant dataset

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
